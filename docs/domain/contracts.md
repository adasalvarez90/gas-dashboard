# Contracts Domain

A contract represents an agreement between the company and an investor.

Contracts manage the lifecycle of the investment and the commissions generated by it.

---

# Contract Lifecycle

A contract does not start when it is signed.

A contract starts only when the first tranche becomes fully funded.


Contract start = fundedAt of first tranche


Contract duration is always:


12 months


Example:


Tranche funded: January 15
Contract end: January 15 next year


---

# Contract Fields

Important fields:


uid
investor
scheme
source
roles
startDate
endDate
contractStatus


---

# Contract Schemes

Two schemes exist:


A
B


These schemes define how commissions are distributed.

---

# Investment Source

The origin of the investment determines commission distribution.

Possible values:


COMUNIDAD
RED_CALIDA
DINERO_PROPIO
REFERIDORA


This value determines which commission matrix is used.

---

# Contract Roles

Each contract assigns advisors to specific roles.

Roles determine how commissions are split.

Example:


consultant
kam
manager
salesDirector
operations
ceo
referral


The referral role is optional and only exists when the source is REFERIDORA.

---

# Contract Cancellation

Contracts may be cancelled.

When this happens:

Future unpaid commissions must be marked as:


cancelled = true


Already paid commissions remain unchanged.
